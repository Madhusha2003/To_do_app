# Upcoming Features - ToDo Buddy Pro

This document outlines the planned enhancements for the application, prioritized from most critical to least critical based on the roadmap for turning this into a "Real Assistant Buddy."

## 🚀 Priority 1: Core Intelligence & Settings
*   **Unified App Settings Page**:
    *   Allow users to toggle between the **Simple Categorizer (ML Model)** and **LLM Categorizer (Nova)**.
    *   Ensure the app remains lightweight and functional without external AI if desired.
*   **Smart Priority Engine**:
    *   Nova automatically analyzes and tags tasks with **High/Medium/Low** priority based on content and dates.
*   **Sub-tasks / Checklists with File Support**:
    *   Add nested items within a task.
    *   **Feature Expansion**: Integrate **RAG (Retrieval-Augmented Generation)** and **LangChain** to allow users to attach files to sub-tasks for context-aware assistance.

## ⚡ Priority 2: Assistant Capability & Access
*   **Global Shortcut for Nova**:
    *   Implement a system-wide hotkey to summon Nova instantly.
    *   **Future Vision**: Ability for Nova to read, access, and add files from your computer to turn it into a true assistant buddy.
*   **Natural Language Search with Context**:
    *   Ask Nova to find tasks using natural language.
    *   **Implementation Note**: Provide the local model with the full context of the task list and calendar to improve accuracy.
*   **Voice-to-Task**:
    *   Microphone button to dictate tasks; AI processes the transcript to extract categories and dates.

## 🛠️ Priority 3: Task Management & Automation
*   **AI Task Decomposition**:
    *   Automatically break down complex tasks (e.g., "Plan Vacation") into actionable sub-steps.
*   **Recurring Tasks**:
    *   Support for tasks that repeat daily, weekly, or monthly.

## 🎨 Priority 4: UX & Polish
*   **Drag-and-Drop Reordering**:
    *   Manually reorder tasks in the list.
*   **Desktop Notifications**:
    *   Real-time system alerts for tasks with specific times.
